The Revendless plugin allows publishers and bloggers to easily integrate the Revendless JavaScript SDK on their website or blog.

== Description ==

The Revendless JavaScript SDK is a toolkit for professional fashion, beauty & lifestyle bloggers and helps them to easily monetize their content.

== Installation ==

Get started:

1. Click the "Activate" link left to the Revendless plugin in your wordpress plugin manager
2. [Sign up](https://www.revendless.com/registration) for a Revendless account and get your API key
3. Go to the Revendless plugin configuration page in Wordpress and submit your personal API key
